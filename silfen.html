Silfen

Silfen is a Danish fashion brand founded in 2013 by Daniel and Celine Silfen which mainly offers bags. They define themselves as playful, innovative and conscious.
Visit website

What was it about

In this group project we had to analyse the brand and redesign their website, so it expresses their values better. The main focus was to create a user-friendly purchase flow. The task was to improve the current one so it was more intuitive as well as organize better the information architecture.

New coding skills

It was for sure the most complex project regarding coding. We created a fully functional website. We used database to store all the information about products. In order to make our website dynamic we fetched all data using JS. I learnt how to manipulate the cart content and I was introduced to the concept of local storage and url params method.
